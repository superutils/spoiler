# Spoiler:
Inspired by telegram's spoilers, it's a React-library that's made to hide anything with a spoiler.

## Example:
![Example](./other/example.gif)

## Installation:

```
npm i
```

`npm run storybook` for testing

## Usage:
```tsx
<Spoiler
  width={/**not required**/}
  height={/**not required**/}
  isActive={/**not required**/}
 >
  <AnyComponent />
</Spoiler>
```
